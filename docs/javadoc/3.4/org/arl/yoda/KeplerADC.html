          • KeplerADC(String type, int channels, String uid, String address)

            Create ADC support agent connecting to specified DAM.

            Parameters:
            type - "LF" (low frequency DAM) or "MF" (mid-frequency DAM).
            channels - number of channels supported by DAM.
            uid - unique ID of DAM (format "0:0:0:0").
            address - IP address and port number of DAM (format "0.0.0.0:0")
