Package: org.arl.unet.state

[Java] Class StateManager

      • org.arl.unet.state.StateManager

      • Agents state persistence manager. This agent allows user to persist state changes, i.e., to save or load the agent states as Groovy scripts. The agent listens to ParamChangeNtf from other agents to keep track of their states, and writes a Groovy script to restore the agent to that state.
