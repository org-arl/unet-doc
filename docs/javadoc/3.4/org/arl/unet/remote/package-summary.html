Package org.arl.unet.remote Description

Remote node control interface and agents.

    RemoteFileNtf Notification of completion of file transfer from a remote node.
    RemoteFilePutReq Request to transfer a file to a remote node.
    RemoteShellExt Remote service shell extension.
    RemoteSuccessNtf Notification to indicate remote command was successful.
    RemoteTextNtf Notification of arrival of text message from remote node.
