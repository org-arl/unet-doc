Package: org.arl.unet.sim.channels

[Groovy] Class Mission2013a

      • org.arl.unet.sim.channels.Mission2013a

      • MISSION 2013a channel model.

        This model is an extension of the protocol channel model with per-link probability of detection and packet loss as measured from the MISSION 2013 experiment (deployment 2).

        The node addresses and locations are also included in the class. Node addresses can be accessed as Mission2013a.nodes, and the (x, y, z) node location in meters is access via the Map Mission2013a.nodeLocation.
