Package org.arl.unet.sim.channels Description

Network simluator channel models.
  • Class Summary 
    Class Description
    AbstractAcousticChannel Base class for implementing acoustic channel models.
    BPSKFadingModel A communication model based on a BPSK frame in an underwater channel modeled as a Rayleigh or Rician fading channel, with a LFM/DSSS detection preamble.
    BasicAcousticChannel A basic underwater communications channel model for simulation.

    Mission2013a MISSION 2013a channel model.

    ProtocolChannelModel Protocol channel model.
