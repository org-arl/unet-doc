Package: org.arl.unet

[Java] Class OutputPDU

        • org.arl.unet.OutputPDU

        • A utility class to help compose PDUs.

          This class extends a ByteArrayOutputStream and adds methods to help
          compose PDUs. The PDU can then be converted to a byte array using the toByteArray() method.
