Package: org.arl.unet

[Java] Class UnetAgent

      • org.arl.unet.UnetAgent

      • Underwater networking stack framework API agent.
        In underwater networking stack, various layers are implemented as agents. fjage provides a Java framework for implementing them. This class provides an agent super-class for all agents in the UnetStack.
