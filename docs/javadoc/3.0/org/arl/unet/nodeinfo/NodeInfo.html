            • public double[] getLocation()

              Gets the node location. The location is in meters from origin and ENU coordinate system, if origin is set. Otherwise it is in decimal [latitude, longitude]. Depth is always in meters. Returns the location based on last update, and speed/heading/turnRate/diveRate if mobility is set to true. Changes in speed/heading/turnRate/diveRate are assumed to be from the time of last location update.

              Returns:
              location as [x, y] or [x, y, z] or null
