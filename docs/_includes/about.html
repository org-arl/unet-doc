What is UnetStack?

A complete toolkit for underwater networks

UnetStack is a collection of technologies to extend communication networks underwater. Built on an agent-based framework, it gives you everything needed to design, simulate, test and deploy underwater networks, from the protocol stack to a software-defined acoustic modem.

01
Design
Compose agents and protocols on a media-agnostic stack.
02
Simulate
Run many-node networks in realistic channel conditions.
03
Test
Validate on real hardware with hardware-in-the-loop.
04
Deploy
Ship the same code to modems at sea. No porting.
The ecosystem

Tools and platforms for the full lifecycle

A growing ecosystem around UnetStack. From desktop simulation to hardware deployment with Subnero modems.

NETWORK SIMULATOR

UnetSim

Fast, flexible simulator with discrete-event and real-time modes. Develop and refine subsea protocols with no code porting.

CHANNEL SIMULATOR

Virtual Acoustic Ocean

Open-source acoustic channel simulator (VAO) interfacing with Bellhop, Kraken and other physics-accurate models for high-fidelity testing.

DEVELOPMENT PLATFORM

UnetCube

A compact UnetStack development platform in the form of a digital modem. Prototype and test your underwater network applications on real embedded hardware.

Learn more →
CLOUD PLATFORM

UnetCloud

Physics-based underwater network simulation hosted in the cloud. Virtual nodes run the same UnetStack software and APIs as real modems, so you can develop and validate protocols without hardware.
